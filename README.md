# Shop Billing System

A simple and efficient Shop Billing System built using Python and SQLite.  
This project helps small shop owners manage products, generate bills, and maintain sales records digitally.

---

## Features

- Add, update, and delete products  
- Generate customer bills  
- Store billing history  
- Calculate total amount automatically  
- Simple and easy-to-use interface  
- Local database using SQLite  

---

## Technologies Used

- Python  
- SQLite  
- File Handling

## How to Run the Project

1. Clone the repository  
   ```
   git clone git@github.com:Uday-ctrl-web/ShopBillingSystem.git
   ```

2. Go into the project folder  
   ```
   cd ShopBillingSystem
   ```

3. Run the program  
   ```
   python main.py
   ```
